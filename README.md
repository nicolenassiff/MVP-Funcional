CONSTRUÇÃO DE UM MVP FUNCIONAL

Esse trabalho será desenvolvido a fim de resolver um problema usual: falta de
organização de tarefas. É muito comum, quando se tem muitas tarefas a se fazer,
esquecer prazos, atividades ou se algo já foi feito. Isso se torna um problema pelo
esquecimento de coisas importantes e atraso de projetos. Por isso, esse trabalho
tem como objetivo estruturar uma solução simples, sem tecnologias avançadas, que
permita o cadastro, a listagem e o acompanhamento de tarefas.
